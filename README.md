## 🎤 Pitch do Projeto
Muitas clínicas veterinárias perdem tempo respondendo as mesmas dúvidas (horários, vacinas, endereço) repetidamente via WhatsApp. O **PetBot Agent** resolve isso automatizando o atendimento inicial 24/7. 

**O resultado?** Uma recepção menos sobrecarregada, clientes com respostas instantâneas e mais tempo para os veterinários focarem no que realmente importa: a saúde dos animais.
